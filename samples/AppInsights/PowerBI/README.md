In this folder, you will find samples that illustrate how you can use Azure Application Insights data in your own Power BI reports.

# Which Power BI reports are available?
This repository contains 
* the source code for the Power BI telemetry app (located in the Reports/AppSource directory)
* sample code and guidance to create your own reports (located in the Build-your-own directory)
